What it does

Audio in, notation out.

Built for a friend who plays many instruments. You drop in any audio or video — mp3, flac, an mp4 screen recording, whatever — and the pipeline does five things:

(1) Strips audio and normalizes it. (2) Runs Demucs to separate the mix into 6 stems (vocals · drums · bass · guitar · piano · other). (3) Picks the loudest non-drum stem as the lead — drums and bleed are dropped. (4) Transcribes that stem to MIDI with Spotify's Basic Pitch. (5) Renders sheet music (PDF + MusicXML via MuseScore) for melodic instruments, or guitar tabs (GP5 + ASCII via a Viterbi DP string/fret allocator) for guitar.

Everything runs on your Mac. No cloud, no API keys, no telemetry. Outputs auto-export to ~/Music/Justin's Magic Music Box/<song>/.

Two knobs

Auto-detect, or force an instrument. Full, or novice-readable.

Focus

Auto looks at every Demucs stem (drums excluded) and picks the loudest as the lead. The pipeline log shows the dB ranking so you can see exactly which instrument won.

  • Auto · default
  • Guitar · forces tabs
  • Violin · forces sparse-source pre-flight

Complexity

Novice mode drops low-confidence and ultra-short notes from Basic Pitch, snaps onsets to a quarter-note grid, biases the tab allocator toward open positions, and uses wider ASCII tab spacing.

  • Full · faithful 16ths + triplets
  • Novice · readable for a beginner
Quick start

Run it on your Mac.

Apple Silicon recommended (uses MPS for Demucs). Initial install is ~6 GB; first Demucs run downloads ~80 MB of model weights on demand.

# 1. one-time: install brew deps
brew install ffmpeg uv just
brew install --cask musescore

# 2. clone + bootstrap
git clone https://github.com/financewithphil/justins-magic-music-box.git
cd justins-magic-music-box
cd app && uv sync --python 3.11 && cd ..
cd workers/demucs && uv sync --python 3.11 --no-install-project && cd ../..
cd workers/basicpitch && uv sync --python 3.11 --no-install-project && cd ../..

# 3. run
just dev      # → http://127.0.0.1:8768

What works well, what doesn't.

Polyphonic-audio-to-tablature is an unsolved research problem. This pipeline gives you a starting point, not a finished score. Always check the MIDI output and edit by hand for anything you'd actually perform.

ScenarioSheet musicGuitar tab
Solo pianoExcellent—
Singer-songwriter (acoustic + voice)GoodOK
Pop / EDMMid (vocal melody fine)Bad
Rock band, dense mixMidMid–bad
MetalVocals onlyBad
Jazz with extended chordsMid (Chordino vocab limited)Mid
